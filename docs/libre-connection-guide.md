# CARENOTE 리브레(FreeStyle Libre) 연동 가이드

리브레 CGM 센서의 연속 혈당을 CARENOTE로 가져오는 방법입니다.
닥터다이어리와 동일한 방식으로, **CARENOTE 클리닉(LibreView 전문가 계정)** 에 센서 데이터를
공유한 뒤 CARENOTE 앱으로 불러옵니다.

---

## 작동 원리 (전체 그림)

```
리브레 센서
   │  (FreeStyle LibreLink 앱이 스캔·업로드)
   ▼
LibreView  ──►  CARENOTE 클리닉(전문가 계정)   ← 여기까지가 "연결"
   │
   ▼
CARENOTE 앱   ← CSV 가져오기(현재) / 자동수집(추후)
```

- **CARENOTE 클리닉**: LibreView "의료전문가(Practice)" 계정. 이름은 `CARENOTE (개인)`.
- 각 사용자는 자기 **리브레 앱에 CARENOTE 클리닉의 연결코드**를 넣어 데이터를 공유합니다.
- 닥터다이어리와 같은 구조 — 애플 건강/단축어 필요 없음.

---

## Part A. (관리자) 사용자 초대하고 연결코드 발급

> CARENOTE 운영자가 LibreView 전문가 계정에서 진행합니다.

1. 브라우저에서 **libreview.com** → **CARENOTE 전문가 계정**으로 로그인
2. 왼쪽 위 **환자 추가(＋ 아이콘)** 클릭 → **환자 초대** 창
3. 입력:
   - **이름 / 성 / 생년월일** — 해당 사용자 정보
   - **환자 연결(병원)**: `CARENOTE (개인)` 그대로
   - **LibreView 계정 → 이메일**: (선택) 사용자의 리브레 계정 이메일
4. **저장** → **연결코드**가 생성됨 (예: `1c288f8432f771aa`)
5. 이 **연결코드를 사용자에게 전달**

---

## Part B. (사용자) 리브레 앱에 연결코드 등록

> 사용자가 본인 아이폰의 리브레 앱에서 진행합니다.

1. **FreeStyle LibreLink** 앱 열기
2. 왼쪽 위 **메뉴(≡)** → **연결된 앱**
3. **Libre Data Share** (의료팀과 공유하는 코드) 선택
4. **연결코드 입력** 칸에 전달받은 코드 입력 (예: `1c288f8432f771aa`)
5. **CARENOTE (개인)** 클리닉이 표시되면 **연결/동의**
6. "LibreView에 연결됨" 표시가 뜨면 완료 ✅

이제 센서가 업로드할 때마다 혈당이 **CARENOTE 클리닉으로 자동 전송**됩니다.

---

## Part C. 혈당을 CARENOTE 앱으로 가져오기

### 현재 방식 — CSV 가져오기 (수동, 확실함)

1. **libreview.com** → CARENOTE 전문가 계정 로그인 → 해당 환자 화면
2. 오른쪽 위 **혈당 데이터 다운로드** → `.csv` 파일 저장
3. **CARENOTE 앱** → **프로필** → **데이터 가져오기** → **📊 리브레뷰** 탭
4. 받은 `.csv` 선택 → 미리보기 "✅ LibreView CSV 데이터 발견" 확인 → **가져오기**
5. 대시보드·혈당 그래프에 연속 혈당이 채워짐

> 며칠에 한 번 CSV를 받아 가져오면 계속 채울 수 있습니다.

### 추후 — 자동수집 (개발 예정)

CARENOTE 서버(Edge Function)가 CARENOTE 클리닉에 자동 로그인해 혈당을 주기적으로
가져오는 방식. LibreView 전문가 API가 비공식이라, 실제 다운로드 요청을 캡처(cURL)해
정확히 구현해야 함. 구현되면 CSV 수동 작업 없이 자동 반영됨.

## 자주 막히는 부분 (문제 해결)

| 증상 | 원인 / 해결 |
|---|---|
| 리브레 앱에 **"연결된 앱 → Libre Data Share"** 가 없음 | 앱을 최신 버전으로 업데이트 후 재확인 |
| **LibreLinkUp** 메뉴가 안 보임 | 한국 리브레2 앱은 LibreLinkUp 미지원 → **Data Share 코드 방식**을 사용(본 가이드) |
| 연결코드를 넣어도 클리닉이 안 뜸 | 코드 오타 확인, 리브레 앱 로그인 상태·국가 설정(대한민국) 확인 |
| 전문가 계정에 환자는 뜨는데 **혈당이 비어 있음** | 센서 다음 업로드까지 대기(수 분~), 리브레 앱에서 센서 스캔 1회 |
| CSV 가져오기에서 **"형식을 인식할 수 없음"** | libreview.com에서 받은 원본 CSV인지 확인 (엑셀로 다시 저장하지 말 것) |

## 참고 — 계정 정리

| 계정 | 용도 |
|---|---|
| **본인 리브레 계정** | 센서가 연결된 원래 LibreView 계정 (FreeStyle LibreLink 로그인 계정) |
| **CARENOTE 전문가 계정** | 데이터를 모으는 클리닉 (`CARENOTE (개인)`), 연결코드 발급·CSV 다운로드 |

> LibreLinkUp 팔로워 계정은 이 방식에선 필요 없습니다.
